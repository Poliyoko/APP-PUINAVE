# SGODA Project Builder 1.0.0

Primera versión estable del Builder SGODA-PUINAVE.

## Capacidades principales

- Inicialización profesional de proyectos SGODA.
- Generación de backend, frontend, database, module, API, workflow y docs.
- Registro automático de componentes.
- Auditoría estructural y de gobierno de datos.
- Reglas DAMA-DMBOK, FAIR y CARE.
- Informes en texto, JSON y Markdown.
- Modo estricto y códigos de salida para CI/CD.
- Empaquetado wheel y source distribution.
- Integración continua multiplataforma.
- Publicación automatizada de GitHub Releases.

## Instalación

```powershell
python -m pip install sgoda_builder-1.0.0-py3-none-any.whl
sgoda version
```
